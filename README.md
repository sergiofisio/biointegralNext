# Biointegral Saúde

Site institucional da clínica Biointegral Saúde — Next.js 15 + Tailwind CSS v4.

## Stack

- **Next.js 15** (App Router)
- **Tailwind CSS v4**
- **TypeScript**
- Deploy: Vercel

## Desenvolvimento

```bash
yarn install
yarn dev
```

Abra [http://localhost:3000](http://localhost:3000).

## Build

```bash
yarn build
yarn start
```

## Estrutura

```
app/           → rotas e páginas
components/    → Nav, Footer, WhatsAppFloat, formulário
lib/           → site-data, seo, utils
public/images/ → assets do site
```
